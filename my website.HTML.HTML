All about scratch and Mit app inventor-
Scratch:

Scratch is a free, visual programming language and online community developed by the MIT Media Lab that allows users, primarily children,to create interactive stories, games, and animations by snapping together colorful code blocks

Mit App Inventor:

MIT App Inventor is a free, web-based platform from the Massachusetts Institute of Technology (MIT) that uses a visual, block-based programming language to allow anyone, including children and beginners, to create functional apps for Android and iOS devices. It simplifies app development by replacing traditional text-based coding withan intuitive drag-and-drop interface, enabling users to design, program, and test apps in one environment.
